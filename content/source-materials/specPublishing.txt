EXECUTIVE OVERVIEW
--------------------------------------------------------------------------------
This specification defines how to transform the AI Agents research project into
a comprehensive publication series for Fernando Torres's personal website. The
research comprises 36 expert interviews, 5 industry conferences, and 3 functional
prototypes—generating rich content that can be repurposed into standalone blog
posts, each accompanied by a LinkedIn promotional post.

THE VISION:
The goal is to establish thought leadership in AI agent deployment through a
systematic content publishing strategy. Each publication will be:
  1. Standalone and valuable without requiring readers to consume other posts
  2. SEO-optimized with clear titles and structured content
  3. Linked to the umbrella research project for credibility
  4. Accompanied by a LinkedIn post driving traffic to the website
  5. Formatted as MDX files compatible with the existing website architecture

WEBSITE ARCHITECTURE ANALYSIS:
--------------------------------------------------------------------------------
The website is a Next.js 14 application using:
  • Contentlayer for content management
  • MDX for rich content with React components
  • Tailwind CSS for styling
  • TypeScript for type safety

CONTENT STRUCTURE:
  /content/
    /blog/           <- Blog posts (MDX files)
    /projects/       <- Project showcases (MDX files)

BLOG POST FORMAT (from contentlayer.config.js):
--------------------------------------------------------------------------------
Required fields:
  • title: string (required)
  • summary: string (required)
  • publishedAt: date (required)

Optional fields:
  • tags: list of strings (default: [])
  • image: string (cover image path)
  • author: string (default: 'Fernando Torres')
  • featured: boolean (default: false)

Computed fields (automatic):
  • slug: generated from file path
  • readingTime: calculated from content
