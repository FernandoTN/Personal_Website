## Design Handoff Notes

### General Guidelines

**Consistency Requirements:**

- All covers should maintain visual cohesion as a series
- Consistent placement of publication number badge (top-left corner)
- Stanford GSB / MSx '26 attribution in bottom margin
- AI Agents Research series identifier

**File Deliverables:**

- Primary: 1200x630px PNG (social sharing)
- Alternative: 1600x900px PNG (presentation slides)
- Source file: Figma/Photoshop with editable layers

**Accessibility:**

- Minimum contrast ratio of 4.5:1 for all text
- Alt text descriptions for screen readers
- Avoid relying solely on color to convey meaning

### Category-Specific Notes

**Prototype Covers (19-21):**

- Emphasize hands-on, practical learning aesthetic
- Include subtle code/technical elements
- Use darker backgrounds for technical credibility
- GitHub/repository subtle branding elements

**Conference Covers (22-24):**

- Professional, summit/event aesthetic
- Include subtle speaker/stage visual elements
- More corporate/polished design language
- Quote callout integration where applicable

**Methodology Cover (25):**

- Academic/research aesthetic
- Clean, systematic, data-visualization style
- Lighter background for readability
- Process-oriented visual flow

### Asset Library Required

**Icons Needed:**

- State machine nodes (6 types)
- E-commerce elements (cart, product, checkout)
- Security icons (lock, shield, key)
- Infrastructure icons (server, database, cloud)
- Agent/bot icons (multiple styles)
- Conference/presentation icons
- Research/document icons
- Trap/warning icons (4 types)
- Model provider logos (GPT, Claude, Gemini) - stylized

**Pattern Library:**

- Circuit board background texture

- Gradient presets for each publication

## Summary

This specification covers 7 unique cover images across three content categories:

**Prototype Publications (19-21):** Technical, hands-on aesthetic with code/building themes emphasizing practical learnings from the Shopping Agent, Repo Patcher, and Good Agents prototypes.

**Conference Publications (22-24):** Professional, insight-focused designs capturing key revelations from an AI autonomous agent company Fireside (30-40% model contribution), the 95% failure conference (four traps and handoff rate), and Production Agents Summit (40% context rule).

**Methodology Publication (25):** Academic, systematic visualization showing the research pipeline from 44 sources to actionable insights.

Each cover maintains series cohesion while visually distinguishing its specific content category and core message.
