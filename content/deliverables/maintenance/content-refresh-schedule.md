# Content Refresh Schedule

## Refresh Frequency by Content Type

| Content Type    | Refresh Frequency | Priority |
| --------------- | ----------------- | -------- |
| Anchor post     | Quarterly         | High     |
| Theme posts     | Semi-annually     | Medium   |
| Interview posts | As needed         | Low      |
| Statistics      | Quarterly         | High     |

## Quarterly Review Checklist

- [ ] Verify all statistics still accurate
- [ ] Update any outdated examples
- [ ] Check for broken links
- [ ] Review for new developments in AI agents
- [ ] Update publication dates if major changes

## Content Aging Indicators

Signs a post needs refresh:

1. Statistics older than 6 months
2. Tools/frameworks mentioned are deprecated
3. Major industry developments not reflected
4. Comments indicate outdated info
